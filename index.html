1) Criar um Repositório no GitHub e criar um HTML simples (pode ser hello word)
2) Suba esse HTML no github pages
3) Você deve criar o Arquivo de Configuração da Pipeline (GitHub Actions) para que cada vez que você faça um commit, a pipeline seja acionada e atualize o seu automaticamente.
a) OBS: o arquivo da action deve obrigatoriamente ser salvo no mesmo repositório onde você criou seu html, dentro das pastas .github/workflows. Por exemplo: .github/workflows/ci.yml
b) https://docs.github.com/pt/actions
c) https://docs.github.com/pt/actions/automating-builds-and-tests/building-and-testing-python
